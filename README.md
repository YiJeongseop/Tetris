## Prerequisites

* It is recommended to create and activate a Python virtual environment
* Install the requirements lybraries with:

  ```bash
  make requirements
  ```

## Run

```bash
make run
```


### Controls

* ↑ Turn the piece
* → Go right
* ← Go left
* ↓ Fall faster
